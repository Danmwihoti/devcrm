# devCRM – Your Friendly Little CRM

A lightweight CRM (Customer Relationship Management) prototype built with Node.js and TypeScript. It demonstrates basic contact management, task tracking, and integration hooks.

## Features
- Add, edit, delete contacts
- Track interactions and notes
- Simple JSON‑based storage (easy to swap for a DB)
- Extensible command‑line interface

## Getting Started
```bash
npm install
npm run dev   # start the development server
```
